# Autostream-Agent


This project demonstrates a conversational chatbot that provides pricing
information and guides users through plan selection.

## Features
- Answers pricing queries
- Explains Basic and Pro plans
- Handles user intent for plan selection
- Collects user details

## Tech Stack
- Python
- Google Generative AI API (if used)

## How to Run
1. Clone the repository
   git clone <repo-link>

2. Install dependencies
   pip install -r requirements.txt

3. Run the application
   python main.py
